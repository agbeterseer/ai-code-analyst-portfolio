# AI Code Evaluation Framework

## Metrics

### 1. Correctness (0–5)

* Does the code meet requirements?

### 2. Efficiency (0–5)

* Time/space complexity

### 3. Safety (0–5)

* Null checks, validation, security

### 4. Maintainability (0–5)

* Readability, modularity
